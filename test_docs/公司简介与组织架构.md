# 公司简介与组织架构

## 公司简介

本公司成立于 2010 年，是一家专注于企业数字化解决方案的高新技术企业，
业务覆盖软件研发、云计算与人工智能等领域，现有员工 800 余人。

## 企业愿景

成为最受客户信赖的企业数字化转型合作伙伴。

## 核心价值观

- 客户第一：始终以客户价值为中心
- 团队协作：开放沟通，互信共赢
- 持续创新：拥抱变化，追求卓越
- 诚信正直：言行一致，恪守承诺

## 组织架构

公司设有以下主要部门：

1. **研发中心**：负责产品设计、开发与测试
2. **市场部**：负责品牌推广与市场拓展
3. **销售部**：负责客户开发与商务签约
4. **人力资源部**：负责招聘、培训与绩效管理
5. **财务部**：负责预算、核算与资金管理
6. **行政部**：负责后勤保障与办公管理
