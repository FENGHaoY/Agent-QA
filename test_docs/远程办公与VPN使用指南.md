# 远程办公与 VPN 使用指南

## 适用场景

本指南适用于因出差、居家办公等原因需要远程访问公司内部系统的员工。

## 申请远程办公

1. 在 OA 系统提交远程办公申请，注明事由与时间段。
2. 经直属上级审批通过后生效。
3. 每月远程办公天数原则上不超过 5 天。

## VPN 使用步骤

1. 从 IT 服务台下载并安装公司指定的 VPN 客户端。
2. 使用企业账号登录，并完成多因素认证（MFA）。
3. 连接成功后即可访问内部办公系统与代码仓库。

## 安全要求

- 远程办公必须使用公司配发的设备，禁止使用个人电脑访问机密数据。
- 公共网络环境下务必通过 VPN 加密通道访问内部系统。
- 离开电脑时应锁定屏幕，防止信息泄露。

## 常见问题

**Q：VPN 连接失败怎么办？**
A：请先检查网络连接，确认账号密码与 MFA 是否正确；若仍无法连接，请联系 IT 服务台。

**Q：远程办公期间如何参加会议？**
A：可通过公司统一的视频会议系统参会，建议提前测试摄像头与麦克风。
